# Prompt for a new chat

Продолжай разработку этого Minecraft-мода как ведущий разработчик.

Сначала полностью прочитай AGENTS.md и все документы, которые он требует.
Проверь git status и существующие изменения. Запусти doctor.cmd. Не начинай
проект заново и не доверяй памяти чата больше, чем файлам проекта.

Продолжи ровно пункт Next action из docs/STATUS.md, реализуй его, проверь
компиляцией и подходящим smoke test, затем обнови STATUS, ROADMAP, CONTENT,
DECISIONS и CHANGELOG там, где это действительно требуется.

Я вайб-кодер: безопасные технические действия выполняй самостоятельно. Спрашивай
только о выборе, который заметно меняет дизайн, лицензию, публикацию или данные
пользователя.
