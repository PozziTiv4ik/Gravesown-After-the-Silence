# Release checklist

- check and build pass from a clean checkout.
- The release JAR is tested in a clean NeoForge 1.21.1 profile.
- Dedicated server smoke test passes.
- No missing textures, models, translations or loot tables.
- Config migration from the previous public version is tested.
- CHANGELOG and docs/STATUS.md are current.
- License, authors, credits and third-party notices are complete.
- No caches, JDKs, worlds, logs or secrets are inside the JAR or Git.
- Publishing requires explicit user approval.
