# Gravesown art workspace

Эта папка хранит редактируемые источники и историю решений. Игра загружает только
финальные ресурсы из src/main/resources/assets/gravesown/.

- concepts/ — выбранные и отклонённые визуальные концепты;
- generation-prompts/ — точные промпты и сведения об инструменте;
- source/ — Blockbench, палитры и другие редактируемые исходники;
- style-guide.md — обязательные визуальные правила.
